## Getting Started

To begin, you'll need to install `console-argument-tracker-loader`:

```console
npm install --save-dev  console-argument-tracker-loader
```

## Usage
development

```console
const a = 1;
console.test(a); // a: 1
// equal
console.log("a:",a); // a: 1

production
console.log and console.test will be removed
```
